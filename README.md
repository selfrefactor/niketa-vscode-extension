# Niketa tools - lightweight Wallaby.js alternative

> VSCode extension

## Start Jest with current file

**Alt+W**

---
if there is such config in package.json
```
  "niketa":{
    "DRAFT.md": "yarn out"
  }
```

then **Alt+W** will run `yarn out` in the current file directory

## Lint current file

**Ctrl+1**
